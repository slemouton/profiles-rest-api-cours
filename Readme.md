- cd profiles-rest-api
- start the development environment (virtual machine) :
	- vagrant up
	- vagrant ssh
- start the virtual environment
	- source ~/env/bin/activate
- go to shared folder
	- cd /vagrant
-start django
	- python manage.py runserver 0.0.0.0:8000
- access the server :
	- http://127.0.0.1:8000
